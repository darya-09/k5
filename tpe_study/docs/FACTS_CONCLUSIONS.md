# ВЫВОДЫ ПО ВСЕМ ЭКСПЕРИМЕНТАМ (только факты, привязанные к числам)

Установка: 50 независимых запусков (seeds), 4 функции × 2 шкалы × 2 типа данных, max_evals=100, gamma=0.2. Значимость — парный Уилкоксон vs baseline `tpe` по seeds с поправкой Холма (α=0.05). Каждый пункт ниже — факт из данных, без интерпретаций.

## Базовые факты
- Средний success_rate: `tpe` = 25.0%, `random` = 4.0%, `optuna` = 21.2%.
- Всего сравнений модификаций с `tpe`: 160. Значимых (Holm): 48; из них улучшений: 42, ухудшений: 6 (все ухудшения — у `random`).

## Какие методы используют точный градиент
- Используют аналитический ∇f (оракул): `tpe_w_smooth`, `tpe_w_smooth_inv`, `tpe_w_sign`, `tpe_w_sign_inv`, `tpe_gp_w`, `tpe_refine`, `tpe_gp_refine`.
- НЕ используют градиент: `random`, `tpe`, `optuna`, `tpe_gp`.

## Факты по градиентному весу (4 формы)
- Суммарно 4 формы w(x) дали 0 значимых улучшений над `tpe`.
- Эти 4 формы не дают значимых улучшений ни при одном из 4 тестов × 3 поправок (см. significance_robust_summary.csv), кроме единичных срабатываний без поправки.
- `tpe_w_smooth`: отличается от `tpe` на 86.8% seeds; медиана разности (algo−tpe) по final_dist_y = 0.0002.
- `tpe_w_smooth_inv`: отличается от `tpe` на 85.0% seeds; медиана разности (algo−tpe) по final_dist_y = 0.0.
- `tpe_w_sign`: отличается от `tpe` на 86.8% seeds; медиана разности (algo−tpe) по final_dist_y = 0.0.
- `tpe_w_sign_inv`: отличается от `tpe` на 82.0% seeds; медиана разности (algo−tpe) по final_dist_y = 0.0.

## Факты по GP и refinement
- `tpe_gp` (без градиента): 6 значимых улучшений; медиана разности по final_dist_y отличается от 0 (см. таблицы).
- `tpe_refine` и `tpe_gp_refine` (локальный спуск по точному ∇f): 12 и 12 значимых улучшений соответственно.

## Факты по нормализации
- Результаты raw и norm СОВПАДАЮТ точно (8/8 ячеек) для: `optuna`, `random`, `tpe`, `tpe_refine`, `tpe_w_sign`, `tpe_w_sign_inv`, `tpe_w_smooth`, `tpe_w_smooth_inv`.
- Результаты raw и norm РАЗЛИЧАЮТСЯ (0/8 совпадений) для: `tpe_gp`, `tpe_gp_refine`, `tpe_gp_w`.

## Что НЕ проверялось (границы применимости фактов)
- Размерность только 2D; один уровень шума на функцию; градиент точный (не зашумлённый).
- GP используется только для переранжирования кандидатов.
- Гиперпараметры фиксированы (gamma, n_candidates, n_init, max_evals); их влияние не варьировалось.
